# J3 Swim Fit Multipage Prototype

This directory contains an alternative multipage take on the J3 Swim Fit website. It keeps the original single-page concept
untouched while expanding content across dedicated pages for discovery, storytelling, and registration.

## Structure

```
site-multipage/
├── assets/
│   └── images/
│       ├── logo.png
│       ├── pool-1.jpg
│       └── pool-2.jpg
├── index.html
├── about.html
├── schedule-faq.html
├── contact-registration.html
├── blog.html
├── scripts/
│   └── main.js
└── styles/
    └── main.css
```

- **index.html** – Home page with hero, programme highlights, approach overview, and blog teasers.
- **about.html** – Team biography, teaching philosophy, and facility highlights.
- **schedule-faq.html** – Class timetable, session structure, and frequently asked questions.
- **contact-registration.html** – Contact information plus an embedded registration form inspired by the original Google Form.
- **blog.html** – Sample posts to demonstrate how ongoing updates could be published.
- **styles/main.css** – Shared design system, layout utilities, and responsive rules.
- **scripts/main.js** – Navigation toggle support and dynamic footer year.

To preview locally, run a static server from the repository root and visit `/site-multipage/index.html`.
